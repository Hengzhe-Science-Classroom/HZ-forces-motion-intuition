# Forces & Motion Intuition · Interactive

**Newton's laws made tangible**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 1 · Foundation

## What You'll Learn

- What is a force? Push, pull, gravity
- Newton's three laws with everyday examples
- Friction: friend and foe
- Speed, velocity, and acceleration
- Simple machines: levers, pulleys, inclined planes
