## Features

- **22 chapters** of interactive physics content across 5 parts
- **Canvas 2D visualizations** with sliders, draggable points, and animations
- **KaTeX** math rendering
- **Dark theme** with progress tracking (saved in localStorage)
- **Zero dependencies** — vanilla HTML/CSS/JS
